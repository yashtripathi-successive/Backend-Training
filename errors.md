# Error Codes Documentation

## HTTP Status Codes

HTTP status codes are issued by a server in response to a client's request. They are grouped by class:

### 1xx: Informational

 Code - Message 

 100  - Continue 
 101  - Switching Protocols 


### 2xx: Success 

 Code - Message 

 200  - OK 
 201  - Created 
 204  - No Content 


### 3xx: Redirection

 Code - Message

 301  - Moved Permanently 
 302  - Found 
 304  - Not Modified 

### 4xx: Client Errors

 Code - Message 

 400  - Bad Request 
 401  - Unauthorized 
 403  - Forbidden 
 404  - Not Found 
 429  - Too Many Requests 

### 5xx: Server Errors

 Code - Message 

 500 - Internal Server Error 
 502 - Bad Gateway 
 503 - Service Unavailable 
 504 - Gateway Timeout
